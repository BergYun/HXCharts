# HXCharts

HXCharts包括了仪表盘、柱状图、圆形图、折线图等四种绘图。

仪表盘、柱状图、圆形图支持渐变色。

柱状图支持水平和竖直两种方向。

仪表盘、柱状图、折线图可以自定义标注值的数量。
